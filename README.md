# Cek Kalori

Scan barcode makanan untuk cek kalori menggunakan data dari [OpenFoodFacts](https://world.openfoodfacts.org/).

**Demo:** [https://cekkalori.vercel.app](https://cekkalori.vercel.app)

## Fitur

- Scan barcode menggunakan kamera HP
- Input kode barcode manual
- Informasi kalori dan nutrisi per 100g
- Riwayat scan tersimpan di localStorage

## Tech Stack

- PHP 8.5 (Vercel Serverless)
- html5-qrcode (barcode scanner)
- OpenFoodFacts API
